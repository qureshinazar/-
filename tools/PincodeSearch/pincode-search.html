What is an Indian PIN Code?

PIN stands for Postal Index Number. It is a 6-digit code used by India Post to identify post offices across India, introduced on August 15, 1972.

PIN Code Structure:

  • 1st digit: Postal region (9 zones)
  • 2nd digit: Sub-region / postal circle
  • 3rd digit: Sorting district
  • Last 3 digits: Specific post office
