  • Bank Passbook — Usually printed on the first page or account details page.
  • Bank Statement — Available on printed or digital bank statements.

🤔 When Do You Need an IFSC Code?

  • NEFT Transfer — Required for National Electronic Funds Transfer.
  • RTGS Transfer — Required for Real Time Gross Settlement (large amounts).
  • IMPS Transfer — Used for Immediate Payment Service transfers.
  • UPI Setup — Some UPI apps may ask for your IFSC during setup.
  • Salary Account — Employers need your IFSC to credit your salary.
  • Tax Refunds — Income Tax Department requires IFSC for direct refund credits.

❓ Frequently Asked Questions

What is an IFSC code?

IFSC (Indian Financial System Code) is an 11-character alphanumeric code assigned by the RBI to every bank branch in India. It uniquely identifies a branch for electronic fund transfers like NEFT, RTGS, and IMPS.

What is the format of an IFSC code?

An IFSC code is 11 characters long: first 4 characters = bank name, 5th character is always 0, and the last 6 characters identify the specific branch.

How do I find my IFSC code?

You can find it on your cheque book, passbook, bank statement, net banking portal, or by using our IFSC Code Finder tool above.

Is this IFSC Code Finder free?

Yes, completely free. No sign-up needed — just enter an IFSC code and get instant results.

Where does the data come from?

We use Razorpay's open-source IFSC API, which maintains an updated database of all Indian bank branches and their IFSC codes.

What if my IFSC code is not found?

If you get an error, double-check the code. Some branches may have been merged or their IFSC codes changed. Contact your bank for the latest IFSC code.
