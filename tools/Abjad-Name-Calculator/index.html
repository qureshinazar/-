📖 About Abjad (Ilm-ul-Adad): Abjad is an ancient numerological system used in Arabic, Persian, and Urdu traditions. Each letter of the alphabet is assigned a numerical value. The total value of a word or name is calculated by summing all its letter values. It is widely used in Islamic numerology, astrology, Taveez writing, and spiritual practices.

❓ Frequently Asked Questions

What is Abjad numerology (Ilm-ul-Adad)?

Abjad is an ancient numerological system that assigns numerical values to Arabic, Persian, and Urdu letters. Each letter has a fixed value from 1 to 1000. The total of all letters in a name or word is its Abjad value.

How do I calculate the Abjad value of a name?

Use the virtual keyboard to type the name in Arabic/Urdu script. The calculator shows the total value in real-time, along with each letter's individual value.

What is the difference between Abjad-e-Kabeer and Abjad-e-Sagheer?

Abjad-e-Kabeer (Greater Abjad) uses standard values from 1 to 1000. Abjad-e-Sagheer (Lesser Abjad) takes the digital root of each value. This calculator uses Abjad-e-Kabeer.

Do Urdu/Persian letters have different values?

Additional Urdu/Persian letters (Pe, Che, Gaf, etc.) take the same value as their closest Arabic equivalent. For example, Pe = 2 (same as Ba), Che = 3 (same as Jeem).

Can I calculate the value of a full name?

Yes! Use the Space button to separate words. The calculator sums all letters across all words of the full name.

Is this tool free and safe?

Yes, completely free. All calculations happen locally in your browser — no data is sent or stored anywhere.
